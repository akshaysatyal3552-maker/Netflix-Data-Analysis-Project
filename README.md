## Description
This project analyzes Netflix's content dataset to uncover trends in movies, TV shows, genres, country distribution, and content growth over time. The goal is to derive actionable insights and visualize key patterns in Netflix content.

## Project Structure
- data/ → raw dataset (`netflix_titles.csv`)
- notebook/ → Jupyter notebook (`netflix_analysis.ipynb`)
- visuals/ → saved chart images
- README.md → project description

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

## Key Insights
1. Movies dominate over TV Shows.
2. USA produces the most content, followed by India and UK.
3. Drama, Comedy, and Documentaries are the most common genres.
4. Netflix content growth increased rapidly after 2015.
5. Most shows/movies are rated PG, TV-MA, or TV-14.
